# charm-nhc

### Introduction
A charm for LBNL Node Health Check (NHC).

### Build and install
```bash
charmcraft build --from ./charm-nhc

# a snap of nhc
wget https://omnivector-public-assets.s3-us-west-2.amazonaws.com/snaps/nhc/edge/nhc_1.4.2-omni_amd64.snap -O nhc.snap
```
